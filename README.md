## Technology
* Kotlin
* Activity for screen host
* ViewGroup
    * RelativeLayout
    * LinearLayout
* UI Components
    - TextView
    - EditText
    - Button
- Toast for message display
- Intent (explicit) to open the 2nd Activity
- FindViewById to retrieve UI Components
- String resource for text concatenation
